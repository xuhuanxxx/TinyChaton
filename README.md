# TinyChaton

![Icon](icon.png)

**TinyChaton** is a lightweight chat enhancement addon for World of Warcraft (Retail), focused on practical features without heavy UI overhead.

## Core Features

- Chat interaction:
  - Sticky channels
  - TAB channel cycling
  - Link hover tooltips
  - Short channel name formats (Short / Number / Number+Short / Full)

- Copy & history:
  - Click timestamp to copy message
  - Per-character local history save/restore
  - Per-channel snapshot selection

- Filters:
  - Blacklist/whitelist by name or keyword
  - Keyword highlight
  - Repeat message filter

- Shelf toolbar:
  - Channel quick switch
  - Quick actions (Ready Check, Roll, Countdown, Emote Panel, Reload, Leave Group, Macro)
  - Theme/size/scale/alpha customization
  - Edit Mode drag positioning

- Emotes & social:
  - Raid icon + custom emote rendering (chat + chat bubbles)
  - Emote panel insertion
  - Auto welcome templates for guild/party/raid

## Installation

1. Download the latest release.
2. Unzip into `World of Warcraft/_retail_/Interface/AddOns/TinyChaton`.
3. Restart game or `/reload`.

## Usage

- Open settings: `/tinychaton` or `Options -> AddOns -> TinyChaton`
- Move shelf: Blizzard Edit Mode
- Use emote panel: shelf emote button (if enabled)

## Localization

Supported locales:
- English (`enUS`)
- Simplified Chinese (`zhCN`)
- Traditional Chinese (`zhTW`)
